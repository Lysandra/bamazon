# bamazon

This application will allow you to make an order from "Bamazon" by selecting a Product ID.  

Bamazon Order:
1.  Enter in Product ID. 

    IF product is available, you will enter the quantity you wish to order and the application will display the total cost.

    ![Image of bamazon_order](https://github.com/Lysandra/bamazon/blob/master/images/bamazon_order.PNG)

    IF product is not available, you will be alerted that we don't carry that Product ID. Please make another selection.

    ![Image of bamazon_product_not_available](https://github.com/Lysandra/bamazon/blob/master/images/bamazon_order.PNG)
